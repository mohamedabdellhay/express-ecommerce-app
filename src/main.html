Welcome to the RESTful API

This API allows you to perform CRUD operations on various resources. Below is a quick guide on how to use it:

Available Endpoints

  • GET /api/v1/products - Retrieve a list of products
  • GET /api/v1/products/:id - Retrieve a single product by ID
  • POST /api/v1/products - Create a new product (send JSON data in the body)
  • PUT /api/v1/products/:id - Update an product by ID (send JSON data in the body)
  • DELETE /api/v1/products/:id - Delete an product by ID
  • GET /api/v1/categories - Retrieve a list of categories
  • GET /api/v1/categories/:id/:title - Retrieve a category products by ID
  • POST /api/v1/categories - Create a new category (send JSON data in the body)
  • PUT /api/v1/categories/:id - Update an category by ID (send JSON data in the body)
  • DELETE /api/v1/category/:id - Delete an category by ID

Example Request

To get all items, make a GET request to:

GET http://localhost:5000/api/v1/products

Authentication

Authorization: Bearer YOUR_TOKEN_HERE

Response Format

The API responds with JSON. Example:

{
    "status": "success",
    "data": {
        "id": 1,
        "name": "Sample Item",
        "description": "This is a sample item."
    }
}

For more information, visit the documentation.
